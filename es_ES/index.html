Comienza a aceptar pagos con Bitcoin con 0 comisiones y sin terceros.

BTCPay Server es un procesador de pagos de criptomonedas, de código abierto y alojado por uno mismo. Es seguro, privado, difícil de censurar y gratuito.

Aprender más   Donar  

Lanza un servidor en la nube, usando tu propio hardware o use un servidor existente.

Conecte su tienda de comercio electrónico o use una de las muchas aplicaciones para recibir pagos.

Los fondos llegarán directamente a tu billetera.

  • Aplicaciones Esenciales Integradas

    Las funciones que más usa se integran directamente en cada instancia para que pueda comenzar rápidamente.

    Cree una aplicación de Punto de Venta para su tienda, inserte fácilmente un botón de propinas en su sitio web, envíe un enlace de pago a clientes de todo el mundo o inicie una campaña de crowdfunding para recaudar fondos para su causa

    Aprender más  

  • Ideal para facturación y contabilidad

    Genera y administra facturas con facilidad.

    Reciba notificaciones cuando les paguen. Exporte sus facturas en diferentes formatos para una contabilidad más cómoda.

    Aprender más  

  • Gestión Integrada de Billetera

    Administre sus fondos entrantes y salientes con una billetera integrada que tiene soporte para billeteras hardware.

  • Servidor auto-gestionado o compartido

    Utilice un servidor compartido si no tiene conocimientos técnicos, o implemente su propia instancia en servidores en la nube o con sus dispositivos físicos.

    Al compartir su servidor, puede apoyar a sus amigos o a su comunidad local.

  • Automación a través del API

    Gestione su servidor BTCPay (por ejemplo, tiendas, facturas, usuarios), automatice sus flujos de trabajo y el control de acesso con nuestra API.

    Aprender más
